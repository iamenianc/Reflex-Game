# Reflex-Game
- A game that tests reaction time
- playable dev version here [text](https://iamenianc.github.io/Reflex-Game/)
- made by i.c.

## Gameplay
Wait for the screen to turn green, then tap as fast as you can — repeat 3–10 times per set. Tap too early and the round fails. Too slow (>1500ms) and it also fails. Your average reaction time is graded from S-tier (under 200ms) down to F. Play solo or challenge a friend in head-to-head mode where the faster average wins the round.
